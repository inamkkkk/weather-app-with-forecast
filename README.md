# Weather App

A simple Flutter weather app that displays the current weather and a 5-day forecast.

## Features

*   Displays current weather information (temperature, description, humidity, wind speed).
*   Shows a 5-day weather forecast.

## Dependencies

*   http
*   provider
*   intl
*   cupertino_icons

## Usage

1.  Clone the repository.
2.  Run `flutter pub get` to install dependencies.
3.  Replace `YOUR_API_KEY` in `lib/services/weather_service.dart` with your OpenWeatherMap API key.
4.  Run `flutter run` to start the app.
